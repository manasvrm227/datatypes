# datatypes
### 1 byte = 8 bit
#### byte : 8 bit | Ranges : 2 <sup>(8 - 1)</sup> to 2<sup>(8 - 1)</sup> - 1 | (-128, 127) 
#### short : 16 bit | Ranges : 2 <sup>(16 - 1)</sup> to 2<sup>(16 - 1)</sup> - 1 | (32768, 32767)
#### int : 32 bit
#### long : 64 bit
#### float : 32 bit
#### double : 64 bit
#### boolean : 1 bit
#### char : 1 bit
